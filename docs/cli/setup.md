# `oni setup`

Initialize `~/.oni/oni.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
oni setup
oni setup --workspace ~/.oni/workspace
```

To run the wizard via setup:

```bash
oni setup --wizard
```
